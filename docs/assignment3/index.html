Menu Bar      Source
Accessibility Feature Access Key Description
Dialer Tab d Use the Dialer tab to access a number pad for entering in and calling a phone number.
Contact List Tab l (lowercase L) The Contact List tab displays the name, email, and phone number of all of the contacts entered into the system.
Add Contacts Tab n Use the Add Contacts tab to add new contact information to the system (name, email, and phone number).
Helper Tab h The Helper Tab, which is the current tab, lists the accessibility features built into this website and details the available accesskeys for quicker site navigation.
Dialer      Source
Accessibility Feature Access Key Description
Digit 0-9 0 through 9 The numeric pads on the Dialer can be accessed by entering in their corresponding number. Ex. the accesskey for the digit "1" is also 1.
Asterisk * * The accesskey for the asterisk * is also an asterisk *.
Pound or Hash Symbol # # The accesskey for the pound sign, also referred to as a hash symbol, is the # (pound sign/hash symbol)
Dial Button e The Dial Button will call the phone number currently listed in the Dialer's phone number entry field. This will only work via a mobile device or other device that supports making phone calls.
Clear Button c The Clear Button will clear out all information in the Dialer's phone number entry field.
